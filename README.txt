Información Tecnica:

JAVA SWING

Requerimientos:
JDK 8 https://www.oracle.com/java/technologies/javase/javase-jdk8-downloads.html (Arquitectura según Sistema Operativo).
innodb_version 5.6+ (Puede instalar XAMPP y activar el servicio de MYSQL) -> https://www.apachefriends.org/es/index.html

Instalación:
asegurarse de tener arrancado el servicio de MYSQL 

Importar en su gestor de base de datos el archivo baseDeDatos.sql que se encuentra en la carpeta de archivos.

Ejecutar el archivo .jar de nombre ejecutable. Este puede ser renombrado y ejecutado desde cualquier lugar de la pc(Escritorio, documentos, etc).

Nota: En caso de que no este en ejecución el servicio de MYSQL la aplicación no pasará mas allá de la inerfaz de inicio.
